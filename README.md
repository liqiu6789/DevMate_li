# DevMate - 智能编程助手

DevMate 是一个 AI 驱动的编程助手，旨在帮助开发者生成和修改代码库。它结合了 MCP (Model Context Protocol) 网络搜索能力和 RAG (检索增强生成) 技术，提供智能化的代码生成与项目构建服务。

## 核心特性

- **全能开发**: 帮助开发者快速构建编程项目及修复问题。
- **知识集成**: 结合实时网络知识 (MCP + Tavily) 与本地文档 (RAG)，提供准确的代码建议。
- **Agent 工作流**: 自主决策何时搜索、何时阅读文档以及何时编写代码。

## 技术栈

- **语言**: Python 3.13
- **环境管理**: uv
- **框架**: LangChain
- **其他**: Docker, MCP, ChromaDB
